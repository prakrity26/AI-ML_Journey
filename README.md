# Titanic Survival Prediction

## Project Overview

This project predicts whether a passenger survived the Titanic disaster using machine learning.

Two classification models were trained:

- Decision Tree Classifier
- Logistic Regression

## Dataset

Titanic Dataset (CSV)

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow

1. Data Loading
2. Data Exploration
3. Data Cleaning
4. Feature Selection
5. Model Training
6. Model Evaluation
7. Feature Importance Analysis

## Results

- Decision Tree Accuracy: 75%
- Logistic Regression Accuracy: 81%

The better-performing model was selected based on test accuracy.
